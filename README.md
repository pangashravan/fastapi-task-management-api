# FastAPI Task Management API

A production-style backend project built with FastAPI.

- Learn scalable backend architecture
- Build recruiter-grade API repo
- Practice clean Git workflow
- Deploy later on Render/Railway

- CRUD APIs
- Pagination + Filtering
- Alembic migrations
- Pytest coverage
- Docker setup
- CI/CD
